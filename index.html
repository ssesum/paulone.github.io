My Github

Hi, I build small applications that may help you. Enjoy.

  1. Print Multiplied Text

    You input text you want to print, and the program will print it how ever many times you want.

  2. Remove New Lines From Text

    Input text where you want to remove all new lines, and the program will print the results.

  3. Convert List To Rows

    Input a comma separated list and convert that to new line separated rows.

  4. Put Quotations Around Text

    Given any amount of comma separated or plain rows, you can put quotations around them.

  5. Python Definition Extraction

    Insert your python code and this program will print out all the rows that contain "def".

  6. Common Value Printer

    Checks two comma separated list for any common terms. Then prints the output to the text area.
